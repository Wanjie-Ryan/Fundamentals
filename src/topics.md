## Notes

- Start with Tier 1 before moving to Tier 2
- Every topic follow this pattern:
  - Understand the concept
  - Code the solution
  - State time and space complexity
  - Know the follow-up optimizations
- Before coding in an interview always:
  1. Clarify the input
  2. Restate the problem
  3. Outline your approach
  4. State complexity upfront
  5. Then code
